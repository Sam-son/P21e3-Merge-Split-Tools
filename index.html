STEP Part 21 Edition 3

The new STEP Part 21 3rd edition allows for simple external referencing in STEP files. A references list at the top of a master file and an anchors list at the top of a child file allows for effective, simple splitting of files at any level of granularity.

Usage

To use, simply run STEPMerge.exe with the name of your master file and the desired output file name.

Options

Optional White- or Black- listing is supported by the addition of the -a or -i flags, respectively.
